# PharmaCare Pro MVP - Developer Notes

This is a specific build for client demonstration.
It works identically to the main repo but has:
- Dynamic mock data (dates/names).
- Simplified configuration.

## How to Run
1.  Install dependencies: `npm install`
2.  Start dev server: `npm run dev`

## How to Build
1.  Run `npm run build`.
2.  The output is in the `dist` folder.
3.  Upload `dist` to Netlify/Vercel manually or via Git.
